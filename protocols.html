Protocols

Iago comes with several protocols built-in and supports easily adding more.

  • HTTP - WWW traffic, have you heard of it?
  • Thrift - cross-language service development from Apache
  • Memcache - A key building block for many scalable online services
  • Kestrel - A queueing system from Twitter
  • UDP - Iago provides the basic building blocks here
